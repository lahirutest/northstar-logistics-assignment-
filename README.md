# NorthStar Logistics Assignment

## Overview
This project analyses operational inefficiencies within NorthStar Logistics using:
- SQL in R
- Statistical analytics in R
- Python operational analytics
- MongoDB NoSQL implementation
- Query optimisation techniques

## Technologies Used
- Python
- R
- SQLite
- MongoDB Atlas
- PyMongo
- pandas
- ggplot2
- seaborn

## Key Findings
- Operational bottlenecks identified at Midtown Relay and Central Core
- Manual route overrides negatively impact customer satisfaction
- Driver behavioural risks identified through anomaly detection
- MongoDB supports scalable operational event storage
- Indexing improves SQL and MongoDB query performance
